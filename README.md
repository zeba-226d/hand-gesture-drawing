Draw in the air using hand gestures tracked by your webcam in real time.

## About

Built as an extension of a hand gesture recognition workshop (volume control project). I expanded the concept into a full drawing application with multiple features.

## Features

- Draw by pinching your fingers together
- 10 color options (red, green, blue, yellow, purple, cyan, white, orange, and more)
- Adjust brush size with + and - keys
- Clear canvas with open palm gesture
- Save your drawing as a PNG with the 's' key
- Quit with 'q'

## Tech Stack

- Python
- OpenCV
- MediaPipe (Hand Landmark Detection)
- NumPy

## How to Run

1. Clone the repo:
git clone https://github.com/zeba-226d/hand-gesture-drawing.git

2. Install dependencies:
pip3 install opencv-python mediapipe numpy

3. Run the app:
python3 air_draw.py

## Controls

| Action | Gesture/Key |
|--------|------------|
| Draw | Pinch fingers |
| Switch color | Point at color palette |
| Clear canvas | Open palm |
| Increase brush | + key |
| Decrease brush | - key |
| Save drawing | s key |
| Quit | q key |
